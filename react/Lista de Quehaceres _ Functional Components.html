Lista de quehaceres

Ah, la lista de tareas! Un proyecto ubicuo para cualquier marco de JavaScript respetable. Tu trabajo es crear una pequeña aplicación React para almacenar y modificar una lista de tareas. Cada tarea tendrá una cadena de texto, así como una propiedad completada, que se establecerá en falso inicialmente. A medida que marque los elementos, deberían aparecer recortados en la página. Usando lo que sabe sobre el estado e iterando a través de listas, renderice una lista de elementos y brinde al usuario la opción de eliminar cada elemento y agregar nuevos. Hay diferentes formas de implementar la función remove. Una forma de hacerlo es utilizando el método de filtro incorporado:  https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter  Filter devuelve una nueva matriz cuando se invoca y es Una excelente manera de mantenerse fiel a la naturaleza funcional de React.

Sugerencia: cuando llames setState, asegúrate de enviar un objeto nuevo.
